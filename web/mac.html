Getting Sunk Coast working on Mac. Thanks to @carpenterjc for making this. I don't own a mac myself, so build is currently unsupported, sorry!

This is the download. You also need homebrew to install the dependencies (glfw and DevIL).

James' instructions for getting the build working, from a command line in the extracted builds directory: 

brew install devil
brew install glfw

chmod 711 sunkcoast
./sunkcoast
